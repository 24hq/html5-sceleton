html5-sceleton
==============

* Install node.js <br>
  node.js -> http://nodejs.org/dist/v0.10.32/node-v0.10.32.tar.gz <br>
  howto -> http://howtonode.org/how-to-install-nodejs

* Install Bower 
  `npm install -g bower`

* Install Grunt 
  `npm install -g grunt-cli`
  
* Prepare locally 
  `npm install && bower install`
  
* Run locally 
  `grunt serve`
